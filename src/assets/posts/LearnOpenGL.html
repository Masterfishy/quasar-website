Learning OpenGL

LearnOpenGL Repo For this project, I am learning OpenGL from this tutorial. Rather than targeting GLAD, however, I've decided to target WebGL because I'd like to host my projects on the web. To compile C++ for WebAssembly and WebGL, I am using Emscripten. I am using Podman containers to develop and host my build artifacts. Not only did I want to learn Podman, but I liked the idea of isolating my dependencies to a container so that I can work between computers and without polluting my WSL images.

Tech Stack

  • C++17
  • CMake
  • Emscripten
  • Podman

Containers

I used two development containers in this project. The first container is the development container I use with VSCode's Dev Containers extension. This creates an image based on the Emscripten image and installs other dev tools. The second container is the hosting container for the WebAssembly artifacts generated by compiling with Emscripten. This image is based off the Nginx image and implements hot reloading like most web development servers (Node.js, Quasar, etc.). The hot reloading uses inotify to monitor file changes in a Podman volume. The development container mounts that volume to the build folder, so when I rebuild my project in the development container, my hosting container detects the file changes relaunches the Nginx server.
